# Cinema
[![CI/CD Pipeline](https://github.com/VickyNipruk/booking-cinema/actions/workflows/ci-cd.yml/badge.svg)](https://github.com/VickyNipruk/booking-cinema/actions/workflows/ci-cd.yml)
Вебзастосунок для бронювання місць у кінотеатрі.
---

## Опис
Cinema — це простий веб-додаток для бронювання місць у кінотеатрі, створений з використанням HTML, CSS та JavaScript.

Користувач може:
- обрати фільм та зал
- вибрати місця
- ввести свої дані
- забронювати квитки

Дані бронювання зберігаються в localStorage браузера.
---

## MVP-ідея
Мінімальний функціонал:
- Відображення місць у залі
- Вибір місць
- Підрахунок вартості
- Введення даних користувача
- Збереження бронювання
- Блокування зайнятих місць
---

## Технології
- HTML5
- CSS3
- JavaScript
- LocalStorage
---
